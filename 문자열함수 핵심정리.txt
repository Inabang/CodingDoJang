strlen => 문자열의길이

strcmp = > 문자열 1과 문자열2를 비교한다 
=> -1 문자열2가 클때
=> 0 같을때
=> 1 문자열1이 클때

Linux와 Mac에서는 문자열1의 ASCII코드와 문자열 ASCII코드를 뺀 결과를 반환한다.
양수면 문자열1이크고 음수면 문자열2 같으면 0

strcpy => 문자열을 복사한다.

strcat => 문자열을붙인다.

spirntf => 서식을 지정하여 문자열을 생성한다.

strchr => 문자열 안에서 문자로 검색한다

strrchr => 뒤에서부터 검색한다

strstr => 문자열안에서 문자열로 검색한다.

strtok => 문자를 기준으로 문자열을자른다.

atoi,atof => 문자열을 정수,실수로 변환

strtol => 특정 진법으로 표기된 문자열을 정수로변환
